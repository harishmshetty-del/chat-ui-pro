# 💬 CodeAlpha Chatbot Pro – Premium Chat UI
A modern ChatGPT-style frontend chatbot UI built using HTML, CSS, and JavaScript. This project focuses on UI/UX design, chat simulation, and frontend development skills.

## ✨ Features
💬 ChatGPT-style UI design  
🎨 Modern dark theme interface  
📱 Fully responsive (mobile + desktop)  
⏰ Message timestamps  
💾 Chat history using localStorage  
💡 Suggested question buttons  
🤖 Rule-based smart responses  
🌙 Dark mode toggle  
📄 Export chat (print to PDF)  
⚡ Smooth animations  

## 🛠️ Tech Stack
HTML5  
CSS3 (Flexbox + Animations)  
JavaScript (DOM + localStorage)

## ⚙️ How to Run Locally
1. Clone repository  
git clone https://github.com/harishmshetty-del/chat-ui-pro.git  
2. Go to folder  
cd chat-ui-pro  
3. Run project  
Open index.html directly in browser  
OR use terminal  
python -m http.server 8000  
Then open  
http://localhost:8000
